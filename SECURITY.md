# Security Policy

## Supported Versions

Security fixes are applied to the latest released version of `agentguard`.

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Instead, email the maintainer at `rss9347@nyu.edu` with:

- A short description of the issue
- Steps to reproduce or a proof of concept
- Any affected versions or environments you tested

I will acknowledge receipt as soon as possible, investigate privately, and coordinate a fix and disclosure timeline with you.
